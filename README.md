<h2 id="summary">簡介</h2>

此作品主要功能有以下:

- 前台
  - 寵物卡片星星點選可收藏 ( 儲存至 localStorage )
  - 捐贈罐頭後點選購物車，完成後續結帳流程。
  - 登入管理後台

- 後台
  - 新增遛噠星人 (可上傳圖片、編輯和刪除)。
  - 新增優惠劵 (可編輯和刪除)。
  - 查看捐贈列表清單
  - 模擬前台捐贈結帳流程
